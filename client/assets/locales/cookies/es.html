COOKIES POLÍTICA

A partir del 11 de julio de 2018.


Rapido US Inc ("Rapido", "nosotros", "nuestro", "nosotros") utiliza cookies en el sitio web www.rapido.com. Nuestra Política de cookies explica qué son las cookies, cómo las utilizamos, la forma en que terceros con los que podemos asociarnos puede usar cookies en nuestro sitio web, sus opciones con respecto a las cookies y más información sobre las cookies.


Al utilizar el sitio web, usted acepta el uso de cookies. Si desea desactivar el uso de cookies, consulte las instrucciones bajo el título a continuación titulado "¿Cómo se pueden desactivar las cookies?"

Qué son cookies?

Las cookies son pequeños fragmentos de texto enviados a su navegador web por un sitio web que visita. Un archivo cookie se almacena en su navegador web y permite que el sitio web o un tercero para que lo reconozca y facilite su próxima visita y el sitio web sea más útil para usted.
Las cookies pueden ser cookies "persistentes" o de "sesión". Las cookies persistentes permanecen en su computadora personal o dispositivo móvil cuando se desconecta, mientras que las cookies de sesión se eliminan tan pronto como cierra su navegador web.

Cómo usamos las cookies?

Cuando utiliza y accede a nuestro sitio web, podemos colocar una serie de archivos de cookies en su navegador web. Utilizamos cookies para habilitar ciertas funciones del sitio web, para proporcionar Analítica para nuestras iniciativas estratégicas y para almacenar sus preferencias.

Galletas de primera

Utilizamos cookies de sesión y persistentes en el sitio web y utilizamos diferentes tipos de cookies para ejecutar el sitio web:

  • Galletas esenciales. Podemos usar cookies esenciales para autenticar a los usuarios y evitar el uso fraudulento de las cuentas de usuario.
  • Cookies de preferencias. Podemos usar cookies de preferencias para recordar información que cambia la forma en que se comporta o se ve el sitio web. como la funcionalidad de "recordarme" de un usuario registrado o la preferencia de idioma de un usuario.
  • Cookies de análisis. Podemos usar cookies de análisis para rastrear la información sobre cómo se usa el sitio web para poder realizar mejoras. También podemos utilizar análisis. Cookies para probar nuevos anuncios, páginas, características o nuevas funciones del sitio web para ver cómo reaccionan nuestros usuarios.
Cookies de terceros

Además de nuestras propias cookies, también podemos utilizar varias cookies de terceros para informar las estadísticas de uso del sitio web y / o entregar anuncios en y a través del sitio web.

How can you disable cookies?

La mayoría de los navegadores aceptan cookies automáticamente, pero le permiten deshabilitarlas. Si bien puede desactivar las cookies, le recomendamos que deje las cookies "activadas". para que podamos ofrecerle una mejor experiencia de compra cuando utiliza el sitio web.
Si desea eliminar las cookies o indicar a su navegador web que elimine o rechace las cookies, visite las páginas de ayuda de su navegador web, tal como se describe a continuación:
Para el navegador web Chrome, visite esta página desde Google:
https://support.google.com/accounts/answer/32050


Para el navegador web Internet Explorer, visite esta página de Microsoft:
http://support.microsoft.com/kb/278835


Para el navegador web Firefox, visite esta página desde Mozilla:
https://support.mozilla.org/en-US/kb/ eliminar-cookies-eliminar-información-sitios web almacenados


Para el navegador web Safari, visite esta página de Apple:
https://support.apple.com/kb/PH21411?locale=en_US


Para cualquier otro navegador web, visite las páginas web oficiales de su navegador web.
Análisis de sitios web de exclusión:
Para inhabilitar el uso de nuestro sitio web en nuestros informes de Google Analytics, visite:
https://tools.google.com/dlpage/gaoptout
